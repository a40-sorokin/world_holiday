Команды GIT

git --version
Версия гита. Как способ узнать, установлен ли он вообще

git init
Иницаилизация локального гита. Создается папка .git

git config 
Параметры пользователя можно задать: имя, еmail ит.п.
git config list - список значений устанвленных параметров

git add <file_name>
добавить в гит указанный файл
git add .
Добавить все неотслеживаемые файлы

git commint -m "comment text"
Создать комит - зафиксировать состояние. До него можно откатиться, с ним можно сравиться.

git status 
показывает какие файлы отслеживаемые, какие нет.
